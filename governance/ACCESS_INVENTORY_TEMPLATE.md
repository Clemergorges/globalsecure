## Objetivo

Manter um inventário mínimo e verificável de quem possui acesso aos sistemas e ambientes da GSS.

## Escopo

- GitHub
- Vercel
- Banco de dados (PostgreSQL)
- Stripe
- Supabase
- Slack (webhook/alertas)
- Pusher
- Provedor cripto/webhooks (Alchemy) e RPC (Polygon)
- SMTP / provedor de e-mail
- Observabilidade (ex.: Sentry)
